# Employee Management System

This is an Employee Management System built using **Spring Boot** (backend) and **React** (frontend).
The application allows users to manage employee details such as name, lastname,email. It provides basic CRUD operations to add, edit, view, and delete employee records.

## Tech Stack

### Backend:
- **Spring Boot**: Java-based framework for building the REST API.
- **Spring Data JPA**: For database interaction.
- **H2 Database** (in-memory database) or any relational database like MySQL/PostgreSQL.

### Frontend:
- **React**: For building the user interface.
- **Axios**: For making HTTP requests to the backend.

## Features

- **CRUD Operations**: Create, Read, Update, Delete employee records.
- **Responsive UI**: The frontend is built to be mobile-friendly using TailWind CSS.

## Prerequisites

Make sure you have the following installed on your machine:
- **JDK 11+**: Java Development Kit.
- **Node.js**: JavaScript runtime.
- **Maven** or **Gradle**: To build and manage Spring Boot dependencies.
- **MySQL/PostgreSQL** (optional if you are not using H2).
- **Git**: Version control system.
